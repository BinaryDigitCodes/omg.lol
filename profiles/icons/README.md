# omg.lol Profile Page Icons

omg.lol uses a combination of [Font Awesome](https://fontawesome.com) and custom icons on profile pages. We’re using the Font Awesome [free collection](https://fontawesome.com/search?m=free), version 6.1.1. The other (custom) icons are part of the omg.lol icon collection.

If you’d like to add an icon or change an icon-domain mapping, feel free to update [icons.md](https://github.com/neatnik/omg.lol/blob/main/profiles/icons/icons.md) with a [pull request](https://github.com/neatnik/omg.lol/pulls).
